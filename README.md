## 📌 프로젝트 소개

이 프로젝트는 자바 프로그래밍의 Term Project로 진행되었으며  
사용자가 감정을 기반으로 일기를 작성하는 애플리케이션입니다.  
Java로 기능을 구현하고 Swing을 통해 GUI를 구현합니다.

감정 선택 및 일기 작성, 일기 열람, 수정 및 삭제, 감정 통계 시각화 등의 기능을 제공합니다.
GUI의 경우 메인 화면 (기능 선택), 일기 작성 화면, 일기 불러오기 화면, 감정 통계 화면으로 구성됩니다.
